# Knuth / Fisher-Yates Shuffle

Implementation of the Fisher-Yates/Knuth shuffle

## Fisher-Yates/Knuth Overview:

Randomly picks integers to swap elements in an ubiased manner.

**Time Complexity:** O(n)

**Space Complexity:** O(n)
